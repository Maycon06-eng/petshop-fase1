# PetShop Amigo Fiel - Projeto Fase 1

## Descrição
Sistema web para um petshop desenvolvido como projeto da disciplina.

## Funcionalidades
- Página inicial com apresentação do petshop
- Página de produtos com 3 categorias:
  - Acessórios (2 produtos)
  - Rações não perecíveis (2 produtos)
  - Produtos de higiene e limpeza (2 produtos)
- Página de serviços com opções de banho e tosa
- Página de contato com informações do estabelecimento

## Tecnologias Utilizadas
- HTML5

## Como Visualizar o Site
Acesse: https://SEU-USUARIO.github.io/petshop-fase1

## Estrutura de Arquivos
- index.html: Página inicial
- produtos.html: Página de produtos
- servicos.html: Página de serviços
- contato.html: Página de contato
